# WynMobile

Aplicativo móvel para conexão entre prestadores de serviços e clientes.

## 🚀 Funcionalidades

- Autenticação de usuários
- Listagem de serviços por categoria
- Detalhes do serviço
- Sistema de avaliação
- Chat entre cliente e prestador
- Integração com Firebase

## 🛠️ Tecnologias

- React Native
- Expo
- Firebase
- GlueStack UI
- React Navigation

## 📋 Pré-requisitos

- Node.js
- npm ou yarn
- Expo CLI
- Android Studio (para emulador Android)
- Xcode (para emulador iOS)

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/WynMobile.git
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o projeto:
```bash
npx expo start
```
